#  Machine: Appointment

Nmap is a tool used to find open ports on a target
- ```sudo nmap {target_IP}```
  - ```sudo nmap -sC -sV {target_IP}```
    - ```-sV``` switch, stands for version detection 
    - ```-sC``` performs a script scan using the default set of scripts
